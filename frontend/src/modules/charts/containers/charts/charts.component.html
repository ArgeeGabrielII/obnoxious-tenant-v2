Third-Party Documentation Available

Chart.js is a third party plugin that is used to generate the charts in this template. This template comes with several custom angular component examples for bar, area, and pie charts. This template also comes with chart service that can provide the chart object to any component. The charts above have been customized - for further customization options, please visit the official Chart.js documentation.
