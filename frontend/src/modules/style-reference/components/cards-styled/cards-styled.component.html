Styled Cards
Waves:
Waves Styled Card
This is an example of a card with the waves styling applied. Use .card-waves to set the background image to the waved styling. This works best with default white cards, and with cards that do not have a card footer.
Angles:
Angles Styled Card
This is an example of a card with the angles styling applied. Use .card-angles to set the background image to the angled styling. This works best with default white cards, and with cards that do not have a card footer.
