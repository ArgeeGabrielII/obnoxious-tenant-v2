Progress Indicator Colors
Bootstrap Defaults:
.bg-primary.bg-secondary.bg-success.bg-info.bg-warning.bg-danger
Use the bgColor attribute to pass a background color utility name to sbpro-progress-bar
Expanded Color System:
.bg-red.bg-orange.bg-yellow.bg-green.bg-teal.bg-cyan.bg-blue.bg-indigo.bg-purple.bg-pink
You can use the expanded color system background utilities to style a progress bar non-contextually.
