Default Bootstrap Borders
Additive:
.border
.border-top
.border-end
.border-bottom
.border-start
The above border utilities will add borders to any element.
Subtractive:
.border-0
.border-top-0
.border-end-0
.border-bottom-0
.border-start-0
The above border utilities will subtract borders from any element that already has them.
