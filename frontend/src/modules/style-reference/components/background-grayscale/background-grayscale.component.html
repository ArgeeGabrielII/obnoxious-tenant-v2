The adaptive grayscale is a custom feature in SB Admin Pro. The grayscale is formed by setting a value to the $grayscale-base-hue variable in the src/styles/sb-admin-pro/variables/_colors.scss file. The base hue, set to blue by default, is then blended with a true black and white grayscale to make a slightly tinted grayscale that is used throughout the theme.

You can use the above utilty classes as background colors for your elements, and you can modify the adaptive grayscale within the src/styles/sb-admin-pro/variables/_colors.scss file.
